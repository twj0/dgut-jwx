- `sfct`: 是否冲突（true=显示冲突课程）
- `szjylb`: 所在教研类别（可选）
- `sfxx`: 是否显示（true=显示）
- `skfs`: 上课方式（可选）
- `kctype`: 课程类型（可选）

**POST 数据** (DataTables 分页格式):

## 🔐 认证机制

### Cookie 认证

**唯一 Cookie**: `bzb_jsxsd`

**示例值**: `66426C0D62471D0DC05D4EEB469F8FAA`

**特点**:
- 所有请求必须携带此 Cookie
- Cookie 在 CAS 登录后获取
- 有效期由服务器控制
- 无需其他 Token 或签名

### 请求头要求

**必需头部**:
```http
Host: jwx.dgut.edu.cn
User-Agent: Mozilla/5.0 (Windows NT 10.0; Win64; x64; rv:148.0) Gecko/20100101 Firefox/148.0
Accept: */*
Accept-Language: zh-CN,zh;q=0.9
Accept-Encoding: gzip, deflate, br, zstd
Cookie: bzb_jsxsd=<SESSION_ID>
```

**AJAX 请求额外头部**:
```http
X-Requested-With: XMLHttpRequest
Referer: https://jwx.dgut.edu.cn/xsxkkc/getGgxxk
```

**POST 请求额外头部**:
```http
Content-Type: application/x-www-form-urlencoded; charset=UTF-8
Origin: https://jwx.dgut.edu.cn
```

## ⚠️ 错误处理与限制

### 1. 学分限制

**错误信息**: "选课失败：系统设定了当前公选课选课学分不能超过3学分！"

**触发条件**:
- 已选公选课学分 + 当前课程学分 > 3

**解决方案**:
- 先退选已有课程
- 选择学分更低的课程

**实际案例**:
```
时间: 20:21:02
尝试选课: kcid=C384FCCB95824EB09F068CE89647F04A, jx0404id=202520262009045
结果: 失败（学分超限）
```

### 2. 其他可能错误

根据系统设计推测的其他错误类型：

- **课程已满**: "选课失败：该课程已满"
- **时间冲突**: "选课失败：上课时间冲突"
- **重复选课**: "选课失败：您已选择该课程"
- **选课未开放**: "选课失败：当前不在选课时间"
- **权限不足**: "选课失败：您没有选课权限"

## 💡 实现建议

### Python 实现示例

```python
import requests

class CourseSelector:
    def __init__(self, session_cookie: str):
        self.session = requests.Session()
        self.session.cookies.set('bzb_jsxsd', session_cookie)
        self.base_url = 'https://jwx.dgut.edu.cn'
        
    def get_course_list(self, page: int = 0, page_size: int = 10, 
                       filters: dict = None) -> dict:
        """获取课程列表"""
        url = f'{self.base_url}/xsxkkc/xsxkGgxxkxk'
        
        # URL 参数
        params = {
            'kcxx': filters.get('kcxx', '') if filters else '',
            'skls': filters.get('skls', '') if filters else '',
            'sfym': 'false',  # 显示未满课程
            'sfct': 'true',   # 显示冲突课程
            'sfxx': 'true'    # 显示
        }
        
        # POST 数据（DataTables 格式）
        data = {
            'sEcho': '1',
            'iColumns': '13',
            'iDisplayStart': str(page * page_size),
            'iDisplayLength': str(page_size),
            'mDataProp_0': 'kch',
            'mDataProp_1': 'kcmc',
            'mDataProp_2': 'xf',
            'mDataProp_3': 'skls',
            'mDataProp_4': 'sksj',
            'mDataProp_5': 'skdd',
            'mDataProp_6': 'xqmc',
            'mDataProp_7': 'xkrs',
            'mDataProp_8': 'syrs',
            'mDataProp_9': 'skfsmc',
            'mDataProp_10': 'ctsm',
            'mDataProp_11': 'szkcflmc',
            'mDataProp_12': 'czOper'
        }
        
        headers = {
            'X-Requested-With': 'XMLHttpRequest',
            'Referer': f'{self.base_url}/xsxkkc/getGgxxk',
            'Origin': self.base_url
        }
        
        response = self.session.post(url, params=params, data=data, 
                                    headers=headers)
        return response.json()
    
    def select_course(self, kcid: str, jx0404id: str) -> dict:
        """选课操作"""
        url = f'{self.base_url}/xsxkkc/ggxxkxkOper'
        
        params = {
            'kcid': kcid,
            'cfbs': 'null',
            'jx0404id': jx0404id,
            'xkzy': '',
            'trjf': '',
            'sfsyjc': ''
        }
        
        headers = {
            'X-Requested-With': 'XMLHttpRequest',
            'Referer': f'{self.base_url}/xsxkkc/getGgxxk'
        }
        
        response = self.session.get(url, params=params, headers=headers)
        return response.json()
    
    def get_selected_courses(self) -> str:
        """获取已选课程"""
        url = f'{self.base_url}/xsxkjg/comeXkjglb'
        response = self.session.get(url)
        return response.text

# 使用示例
selector = CourseSelector('66426C0D62471D0DC05D4EEB469F8FAA')

# 获取课程列表
courses = selector.get_course_list(page=0, page_size=10)

# 选课
if courses['aaData']:
    course = courses['aaData'][0]
    result = selector.select_course(
        kcid=course['kcid'],
        jx0404id=course['jx0404id']
    )
    print(result['message'])
```

### 关键注意事项

1. **必须携带正确的 Cookie**
   - Cookie 名称: `bzb_jsxsd`
   - 从 CAS 登录获取

2. **必须设置正确的请求头**
   - `X-Requested-With: XMLHttpRequest`（AJAX 请求）
   - `Referer` 和 `Origin`（POST 请求）

3. **处理学分限制**
   - 选课前检查当前学分
   - 捕获学分超限错误

4. **实现重试机制**
   - 网络错误重试
   - 课程已满时轮询

5. **遵守选课规则**
   - 检查时间冲突
   - 避免重复选课
   - 尊重选课时间限制

## 🔍 总结

### 核心发现

1. **选课机制简单直接**
   - 单一 Cookie 认证
   - GET 请求即可选课
   - 无复杂加密或签名

2. **学分限制是主要障碍**
   - 公选课总学分不能超过 3 分
   - 需要先退选才能选新课

3. **API 响应快速**
   - 平均响应时间 < 100ms
   - 适合自动化抢课

4. **数据结构清晰**
   - JSON 格式规范
   - 字段命名有规律

### 实现难点

1. **CAS 登录获取 Cookie**
   - 需要处理 CAS 认证流程
   - Cookie 有效期管理

2. **学分计算**
   - 需要先查询已选课程
   - 计算剩余可选学分

3. **并发控制**
   - 热门课程需要抢课
   - 需要实现快速重试

4. **错误处理**
   - 多种失败情况
   - 需要友好的错误提示
