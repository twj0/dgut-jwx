# 选课 API 技术实现文档

## 核心 API 端点

### 1. 课程列表查询 API

**端点**: `POST /xsxkkc/xsxkGgxxkxk`

**完整 URL**:
```
https://jwx.dgut.edu.cn/xsxkkc/xsxkGgxxkxk?sfym=false&sfct=true&sfxx=true
```

**请求头**:
```http
Content-Type: application/x-www-form-urlencoded; charset=UTF-8
X-Requested-With: XMLHttpRequest
Cookie: bzb_jsxsd=<SESSION_ID>
Referer: https://jwx.dgut.edu.cn/xsxkkc/getGgxxk
```

**POST 参数** (用于分页):
```
sEcho=5
iColumns=13
iDisplayStart=0    # 起始位置，0表示第一页，10表示第二页
iDisplayLength=10  # 每页显示数量
mDataProp_0=kch
mDataProp_1=kcmc
mDataProp_2=xf
mDataProp_3=skls
mDataProp_4=sksj
mDataProp_5=skdd
mDataProp_6=xqmc
mDataProp_7=xkrs
mDataProp_8=syrs
mDataProp_9=skfsmc
mDataProp_10=ctsm
mDataProp_11=szkcflmc
mDataProp_12=czOper
```

**响应结构**:
```json
{
  "aaData": [课程列表],
  "sEcho": "5",
  "iTotalRecords": 46,           // 总记录数
  "iTotalDisplayRecords": 46,    // 显示记录数
  "jx0502zbid": "...",
  "flag1": true,
  "jfViewStr": ""
}
```

## 2. 选课操作 API

**端点**: `GET /xsxkkc/ggxxkxkOper`

**完整 URL**:
```
https://jwx.dgut.edu.cn/xsxkkc/ggxxkxkOper?kcid={kcid}&cfbs=null&jx0404id={jx0404id}
```

**请求头**:
```http
X-Requested-With: XMLHttpRequest
Cookie: bzb_jsxsd=<SESSION_ID>
Referer: https://jwx.dgut.edu.cn/xsxkkc/getGgxxk
```

**查询参数**:
- `kcid`: 课程ID (从课程列表获取)
- `cfbs`: 冲突标识 (通常为 `null`)
- `jx0404id`: 教学班ID (从课程列表获取)

**成功响应**:
```json
{
  "success": true,
  "message": "选课成功",
  "jfViewStr": ""
}
```

**失败响应示例**:
```json
{
  "success": false,
  "message": "选课失败：人数已满",
  "jfViewStr": ""
}
```

## 实现示例

### Python 实现 (使用 httpx)

```python
import httpx
from typing import List, Dict, Optional

class CourseSelector:
    def __init__(self, session_cookie: str):
        self.base_url = "https://jwx.dgut.edu.cn"
        self.headers = {
            "User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64; rv:148.0) Gecko/20100101 Firefox/148.0",
            "X-Requested-With": "XMLHttpRequest",
            "Referer": f"{self.base_url}/xsxkkc/getGgxxk",
            "Cookie": f"bzb_jsxsd={session_cookie}"
        }

    def get_courses(self, start: int = 0, length: int = 10) -> Dict:
        """获取课程列表"""
        url = f"{self.base_url}/xsxkkc/xsxkGgxxkxk"
        params = {
            "sfym": "false",
            "sfct": "true",
            "sfxx": "true"
        }

        data = {
            "sEcho": "1",
            "iColumns": "13",
            "iDisplayStart": str(start),
            "iDisplayLength": str(length),
            "mDataProp_0": "kch",
            "mDataProp_1": "kcmc",
            "mDataProp_2": "xf",
            "mDataProp_3": "skls",
            "mDataProp_4": "sksj",
            "mDataProp_5": "skdd",
            "mDataProp_6": "xqmc",
            "mDataProp_7": "xkrs",
            "mDataProp_8": "syrs",
            "mDataProp_9": "skfsmc",
            "mDataProp_10": "ctsm",
            "mDataProp_11": "szkcflmc",
            "mDataProp_12": "czOper"
        }

        with httpx.Client() as client:
            response = client.post(url, params=params, data=data, headers=self.headers)
            return response.json()

    def select_course(self, kcid: str, jx0404id: str) -> Dict:
        """选课操作"""
        url = f"{self.base_url}/xsxkkc/ggxxkxkOper"
        params = {
            "kcid": kcid,
            "cfbs": "null",
            "jx0404id": jx0404id
        }

        with httpx.Client() as client:
            response = client.get(url, params=params, headers=self.headers)
            return response.json()

    def filter_available_courses(self, courses: List[Dict],
                                 max_credits: float = 1.0) -> List[Dict]:
        """筛选可选课程"""
        available = []
        for course in courses:
            # 检查学分
            if course.get("xf", 0) > max_credits:
                continue

            # 检查剩余人数
            syrs = int(course.get("syrs", "0"))
            if syrs <= 0:
                continue

            # 检查冲突
            if course.get("cfbs") is not None:
                continue

            available.append(course)

        return available

# 使用示例
selector = CourseSelector("YOUR_SESSION_COOKIE")

# 获取所有课程
result = selector.get_courses(start=0, length=50)
courses = result["aaData"]

# 筛选1学分的可选课程
available = selector.filter_available_courses(courses, max_credits=1.0)

# 选择第一个可用课程
if available:
    course = available[0]
    print(f"选择课程: {course['kcmc']} ({course['xf']}学分)")

    result = selector.select_course(
        kcid=course["kcid"],
        jx0404id=course["jx0404id"]
    )

    if result["success"]:
        print(f"✓ {result['message']}")
    else:
        print(f"✗ {result['message']}")
```

## 常见问题

### Q: 如何获取所有课程？
A: 使用分页查询，`iDisplayStart` 从 0 开始递增：
```python
all_courses = []
start = 0
length = 50

while True:
    result = selector.get_courses(start, length)
    courses = result["aaData"]
    all_courses.extend(courses)

    if len(courses) < length:
        break  # 已获取所有课程

    start += length
```

### Q: 如何判断课程是否有时间冲突？
A: 检查 `cfbs` 字段和 `ctsm` 字段：
```python
if course.get("cfbs") is not None or course.get("ctsm"):
    print(f"课程冲突: {course.get('ctsm', '未知原因')}")
```

### Q: 选课失败的常见原因？
A:
1. 人数已满 (`syrs` = 0)
2. 学分超限 (总学分 > 3)
3. 时间冲突 (`cfbs` 不为 null)
4. Cookie 过期 (需重新登录)
5. 选课时间未开放

## 总结

### 选课流程
1. 调用课程列表 API 获取所有课程
2. 筛选符合条件的课程 (学分≤1, 剩余人数>0, 无冲突)
3. 调用选课 API 进行选课
4. 验证选课结果

### 关键参数
- **kcid**: 课程唯一标识
- **jx0404id**: 教学班标识
- **xf**: 学分 (当前只能选1学分)
- **syrs**: 剩余人数 (必须>0)
- **cfbs**: 冲突标识 (必须为null)
